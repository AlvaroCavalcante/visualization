# EasyHouse: Plataforma para análise imobiliária e previsão de preços

![Easy house image](assets/easy_house.png)

Só quem já precisou encontrar um casa ou apartamento para alugar sabe o quão difícil essa tarefa pode ser. Inúmeras opções de imóveis disponíveis e fatorem que podem influenciar em seu valor, entre elas o número de quartos, metros quadrados, endereço e muitos outros.

Pensando nisso, o Easy House é uma plataforma em estágio inicial pensada para ajudar você a encontrar o seu novo lar. O processo é muito simples, basta sinalizar uma cidade e estado e deixar a gente fazer o resto! 

A plataforma do Easy House coleta automaticamente os anúncios de imóveis em sua cidade, processa os dados e sumariza isso de maneira simples e intuitiva em um relatório WEB!

## Como instalar

Para instalar as dependências do projeto, basta executar o seguinte comando:
```
pip install -r requirements.txt
```

## Como executar o projeto
Para executar o projeto, execute o seguinte comando na raiz desse repositório:
```
python3 house_bot.py
```
Ao final do processo, o relatório informativo deve aparecer na pasta de *"reports"* na raiz do projeto.
